# ⚡ The Third Strike - Anti-Kuddus Protocol

> **"Three complaints. Two warnings. One final strike."**

## 📖 About
The Third Strike is a student safety and accountability platform built for the **BAIUST Computer Club Hackathon 2026**. It allows students to anonymously report misconduct, send SOS alerts, track corrupt activities, and ensure captain accountability through a democratic strike system.

## 🚀 Features

### 🎯 Core Features
- **Student Dashboard** - Track strikes, warnings, and class activities
- **Teacher Dashboard** - Review complaints and issue strikes
- **Captain Dashboard** - Monitor SOS alerts and student safety

### 📋 Missions
1. **Anonymous Whistleblower** - Secure, anonymous complaint submission
2. **Anti-Camouflage Seat Planner** - Algorithmic seating to catch Kuddus
3. **Syllabus AI Summarizer** - AI-powered syllabus simplification
4. **Corrupt Economy Ledger** - Track Kuddus's financial crimes
5. **SOS Rescue Flare** - Real-time emergency alerts
6. **Kuddus Fact-Checker** - Debunk fake rules instantly

### 🎨 UI Features
- 🌙 Dark/Light Theme
- 📱 Fully Responsive
- ✨ Micro-Interactions & Animations
- 🎯 Glassmorphism Design

## 🛠️ Tech Stack

### Frontend
- **Framework**: React.js + Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Routing**: React Router DOM
- **State**: React Context API

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB (Atlas)
- **ORM**: Mongoose
- **Auth**: JWT + bcryptjs
- **Real-time**: Socket.io
